# p4a.xyz

Static multi-page prototype for the People's Purple Protopian Party of Australia.

## Portal + First 10 Rooms

The homepage is the portal. The first ten rabbit-hole rooms are:

1. It Started as a Joke on a Joint
2. The Purple Synthesis
3. The Fair Go Evolves
4. The Braided Economy
5. The Sovereignty Stack
6. The Truth Engine
7. Civic Surges
8. Minjerribah to Gold Coast Pipeline
9. The Cyber-Republic Simulator
10. Deployment Gear
11. Musicverse

## Colour Direction

The site now uses royal purple as the dominant environment, with amethyst neon, plasma magenta, solarpunk green and signal gold for navigation through the rabbit hole.
